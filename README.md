## Run Locally

Clone the project

Убедитесь, что установлен dotnet 7: https://dotnet.microsoft.com/en-us/download/dotnet/7.0

Если необходимо поменять ip и порт: Properties -> launchSettings.json -> http profile

Для запуска:

Install dependencies
  dotnet restore

Start the server
  dotnet run
